# Prompt Engineering Notes — PrismaEdu

Este arquivo documenta as decisões e técnicas de prompt engineering utilizadas no projeto.

## Técnicas Aplicadas

### 1. Persona Prompting
Definição do papel do professor no system prompt para contextualizar o comportamento do modelo.

### 2. Context Setting
Injeção do perfil do aluno (nome, idade, nível, estilo de aprendizado) no prompt para personalização.

### 3. Chain-of-Thought
Solicitação de raciocínio passo a passo nas explicações conceituais para maior clareza didática.

### 4. Output Formatting
Estrutura esperada da resposta definida no prompt para garantir consistência.

### 5. Few-Shot
Ao menos um exemplo incluído no prompt de exemplos práticos para guiar o modelo.
